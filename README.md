# TuxQuiz, a Linux/Unix command quiz

---

This project is made as a final project for CPE100 - Computer Programming for Engineers, semester 1 academic year 2025.

  ## 1. Introduction
  The goal of this project is to assist users in reviewing key Linux ideas using a command-line quiz program. A series of multiple-choice questions with four possible answers (a, b, c, and d) are presented. Each input is reviewed by the program, which provides prompt feedback. This project is meant for novices who wish to practice their Linux skills in an easy and engaging way.

  ---

  ## 2. Objectives
  - Create a simple test quiz program for Linux learners
  - Practice using the C programming language for console-based applications
  - Apply programming concepts such as loops, conditionals, and functions
  - Use GitHub for version control, project organization, and documentation

  ---

  ## 3. Tools and Technologies
  - C Programming Language – used to build the quiz logic
  - Git and GitHub – used for project hosting and version control

  ---

  ## 4. System Overview
  The program runs in the terminal. Each question includes four options: a, b, c, and d. Users select an answer by typing the corresponding letter. The program validates the input, checks whether the response is correct, and updates the score. At the end of the quiz, the program displays the total score.

  ---

  ## 5. Key Features
  - Multiple-choice Linux questions
  - Simple text-based interface
  - Quick feedback on answers
  - Score summary after completing the quiz
  - Beginner-friendly design
